# InundaTumaco
Sistema web para el reporte y monitoreo de inundaciones en Tumaco

---

##  Descripción
InundaTumaco es una aplicación web que permite a los ciudadanos reportar calles inundadas durante lluvias, principalmente causadas por fallas en el alcantarillado o drenaje. La plataforma centraliza estos reportes para visualizar zonas críticas, hacer seguimiento y facilitar la toma de decisiones por parte de la comunidad y entidades responsables.

---

##  Objetivo
Brindar una herramienta digital que permita identificar zonas afectadas por inundaciones en tiempo real y mejorar la respuesta ante emergencias.

## Problema que resuelve
Actualmente no existe un sistema organizado para reportar y monitorear inundaciones en tiempo real. Esto genera:
- Falta de información centralizada  
- Respuesta lenta de las autoridades  
- Riesgos para la comunidad  

---

##  Funcionalidades principales
- Registro de reportes de inundaciones  
- Visualización en mapa de zonas afectadas  
- Listado de reportes  
- Seguimiento del estado de cada reporte  
- Comentarios y actualizaciones
